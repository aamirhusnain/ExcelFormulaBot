
# ExcelFormulaBot Add-in (AI Formula Generator for Excel)

ExcelFormulaBot is an AI-powered Excel Add-in that helps users generate, understand, and optimize Excel formulas instantly. Simply describe what you want in plain English, and the add-in generates the correct formula for you.

## ⚙️ How It Works

1. Open Excel and select a cell  
2. Launch the ExcelFormulaBot Add-in  
3. Describe your requirement (e.g., “Calculate total sales if region is US”)  
4. AI generates the correct formula  
5. Insert the formula into the selected cell  
6. Optionally get an explanation of the formula  

---

### 5. Sideload Add-in into Excel

* Open Excel
* Go to Insert → Add-ins → Upload My Add-in
* Upload the manifest file
